## 🎯 MVP 优先级 (第一版本)
1. **核心查词功能** - 支持 2-3 种主流词典格式
2. **基础 UI** - 简洁现代的查词界面
3. **生词本** - 基础的收藏和复习功能
4. **跨平台支持** - 至少支持 Windows/macOS/Android
5. **离线使用** - 无需网络的基础功能
